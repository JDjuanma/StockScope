StockScope Análisis simple de empresas y cotizaciones

Proyecto académico

Elegí qué análisis querés hacer.

StockScope conecta una interfaz Bootstrap con una API Express y una base SQLite.

Cómo usarlo

Primero seleccioná una función. Después se despliega solo el formulario necesario.

Funciones disponibles

Seleccioná una opción para empezar.

Buscar empresa

Buscá una empresa por nombre y revisá sus datos principales.

Ver evolución

Consultá precio inicial, precio actual y variación porcentual.

Comparar empresas

Compará dos empresas por su variación en el período cargado.

Ranking

Ordená empresas por una métrica del servicio.

Filtrar empresas

Filtrá por sector, país, precio mínimo, precio máximo o variación.
