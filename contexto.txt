# Contexto del proyecto StockScope

## Stack
- Node.js + JavaScript
- Express
- SQLite con better-sqlite3
- HTML/CSS/JS + Bootstrap
- node:test para pruebas automatizadas

## Arquitectura en capas
- `src/repository/` → conexión a la base de datos y consultas SQL
- `src/services/` → lógica de negocio, cálculos y validaciones simples
- `src/server.js` → API Express y servidor de archivos estáticos
- `public/` → interfaz web con Bootstrap, CSS propio y JavaScript con fetch
- `tests/` → pruebas automatizadas del servicio y datos semilla

## Funciones completadas
- `buscarEmpresa(nombre)` ✅ devuelve datos principales, sector y país de origen
- `verEvolucion(nombre)` ✅ devuelve cotizaciones + resumen estadístico
- `compararEmpresas(nombre1, nombre2)` ✅ devuelve ambas evoluciones + ganadora
- `rankear(criterio)` ✅ criterios: variacion, precio_actual, volumen, volatilidad
- `filtrar(parametro, valor)` ✅ parámetros: sector, pais, precio_min, precio_max, variacion_positiva, variacion_negativa

## API Express completada
- `GET /api/empresa/:nombre`
- `GET /api/evolucion/:nombre`
- `GET /api/comparar?empresa1=...&empresa2=...`
- `GET /api/ranking/:criterio`
- `GET /api/filtrar?parametro=...&valor=...`

## Interfaz web completada
- `public/index.html` → página principal con selector de funciones y secciones desplegables
- `public/styles.css` → estilos propios, foco visible y correcciones de contraste
- `public/app.js` → consumo de API con fetch, render de resultados y lógica de visibilidad

## Accesibilidad y UX
- WAVE revisado y corregido
