# Contexto del proyecto StockScope
## Stack
- Node.js + JavaScript (backend)
- SQLite con better-sqlite3
- HTML/CSS/JS (frontend, próxima etapa)

## Arquitectura en capas
- src/repository/ → habla con la base de datos
- src/services/   → lógica de negocio y cálculos
- src/routes/     → recibe pedidos web (próxima etapa)
- public/         → interfaz de usuario (próxima etapa)
